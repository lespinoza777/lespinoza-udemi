# lespinoza-udemi
UDEMI

Piedra, Papel o Tijera


Este proyecto consiste en el desarrollo de un juego de Piedra, Papel o Tijera utilizando Python.

El programa permite que un usuario juegue contra la computadora, la cual genera una elección aleatoria en cada ronda.

Funcionalidades
Selección de Piedra, Papel o Tijera.
Generación aleatoria de la jugada de la computadora.
Determinación automática del ganador.
Detección de empates.
Sistema de repetición de partidas.
Marcador de puntuaciones.
Herramientas utilizadas
Python 3
Visual Studio Code
GitHub
