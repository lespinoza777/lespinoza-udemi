
Piedra, Papel o Tijera

 Descripción

Este proyecto consiste en el desarrollo de un juego de **Piedra, Papel o Tijera** utilizando el lenguaje de programación **Python**.

El programa permite que un usuario juegue contra la computadora. La computadora genera una elección aleatoria y el programa compara ambas opciones para determinar el resultado de cada partida.


Objetivo

Desarrollar un programa que permita aplicar los conceptos básicos de lógica de programación, utilizando estructuras condicionales, estructuras repetitivas y validación de datos.

Funcionalidades

* Selección de piedra, papel o tijera.
* Generación aleatoria de la jugada de la computadora.
* Comparación entre las elecciones.
* Determinación del ganador.
* Sistema de puntuación.
* Opción para jugar varias partidas.


 Tecnologías utilizadas

* Python 3
* Visual Studio Code
* GitHub




Cómo ejecutar el programa

1. Abrir el archivo `piedra_papel_tijera.py` en Visual Studio Code.
2. Ejecutar el programa.
3. Escribir una de las siguientes opciones:

   * piedra
   * papel
   * tijera
4. Observar el resultado de la partida.
5. Elegir si desea jugar nuevamente.
